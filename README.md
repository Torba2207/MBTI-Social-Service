## MBTI Social Service

**MBTI Social Service** is a web-based social network built around the popular Myers-Briggs Type Indicator (MBTI).  Discover, connect, and engage with people who share your personality type, exploring common interests, values, and perspectives. 

## Key Features

* **Type-Based Discovery:** Find users based on their MBTI type, fostering meaningful connections with like-minded individuals.
* **Personalized Content:** Explore curated content, discussions, and events tailored to your type's strengths and preferences.
* **Community Engagement:** Join type-specific groups, participate in discussions, and share your experiences with others.
* **Type Insights:** Learn more about your own type and how it interacts with others, enhancing self-awareness and understanding.

## Technology Stack

* **Frontend:** React, Next.js
* **Backend:** Java, Spring, Spring Boot
* **Database:** PostgreSQL
* **Caching:** Redis

## Purpose

* **Connect with your type:** Find your people and build meaningful relationships.
* **Explore your type:** Learn more about your personality and how it influences your world.
* **Engage in meaningful discussions:** Share your thoughts and ideas with a community that understands you.

## Running the Project
* **Clone the repository:**
``` https://github.com/Torba2207/MBTI-Social-Service ```
* **Install dependencies:**
``` Docker ```
* **Run the project:**
``` docker-compose up --build -d```
* **Access the application:**
* * **Frontend:** ``` http://localhost:8081 ```
* * **Backend:** ``` http://localhost:8080 ```
* * **PostgreSQL** and **Redis** will be running in the background within the Docker containers.
* **Access the backend documentation:** ``` http://localhost:8080/v3/api-docs ```
* **Stoping the project:**
``` docker-compose down ```
